시력강화 훈련

격자 위를 이동하는 도형을 눈으로 추적하는 훈련입니다. 3단계(가로→세로→랜덤)로 구성되며 도형·점멸·속도를 조절할 수 있습니다. 각 단계가 끝나면 10초 휴식 후 자동으로 다음 단계가 시작됩니다.

집중력 훈련

격자 위를 이동하는 낱말을 읽으며 반복 등장하는 단어를 기억하는 훈련입니다. 3단계로 구성되며 훈련 후 기억한 단어를 적어 봅니다. 수준(하/중/상)에 따라 반복 단어 수가 달라집니다.

기억력 훈련

화면 여러 위치에 낱말이 순서대로 나타납니다. 10라운드 3회차 반복으로 기억력을 강화합니다. 훈련 후 어느 위치에서 어떤 단어가 나왔는지 적어 봅니다.

문장기억 훈련

여러 문장이 글바탕에 낱말씩 순서대로 점멸합니다. 10라운드 진행 후 정답을 확인합니다. 수준(하=1낱말, 중=2낱말, 상=3낱말씩)과 문장 수(2~6개)를 조절할 수 있습니다.

공통 기능

  • 속도 슬라이더로 훈련 속도 조절
  • ⛶ 또는 F11로 전체화면 전환
  • 마침 버튼으로 현재까지 결과 확인

데이터 팩이란?

기본(내장) 외에 추가 낱말·문장 데이터를 불러와 훈련에 활용할 수 있습니다. 한 번 저장하면 인터넷 없이도 영구 사용 가능합니다.

데이터 팩 받는 방법

1

하단 데이터 ▲ → 데이터 팩 연동 클릭

2

데이터 팩 URL을 입력란에 붙여넣기

3

불러오기 & 저장 클릭 → 완료 메시지 확인

4

데이터 ▲ 목록에서 새 팩 선택 후 훈련 시작

사용 가능한 팩

  • 한글_일반 — 일반 한국어 낱말·문장
  • 성경_창세기 ~ 성경_요한계시록 — 성경 66권 개별 팩

JS 집중력·기억력 향상훈련

시력강화·집중력·기억력·문장기억 4가지 뇌 훈련을 제공하는 무료 앱입니다.

개발 목적

성경 말씀 읽기 능력 향상 및 인지 능력 강화를 통해 모든 세대의 두뇌 건강에 기여하고 선교 사역을 지원합니다.
